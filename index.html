Rogue Neo

Here we have my third-year project called Rogue Neo, it's a stealth game with a strong focus on AI and behavior trees. The game is set in a world where the player is trying to steal data from a corporation. The game is made in Unreal Engine 4 and uses C++ and Blueprints. Post Processing

In this project, I was introduced to post-processing effects. This involved rendering a scene to a texture instead of the back buffer, creating effects, and then copying the textures to the backbuffer to show on the screen. In simpler terms, post-processing is used to change the scene in order to reach a desired effect utilising the GPU, similar to adding filters to a photo. This project involved, setting up a 3D scene, working with pixel shaders and vertex shaders to manipulate the pixel colours to achieve a specific effect using DirectX. I had implemented 7 effects, and also implemented area post processing, which enabled the effects in a specific area, in this case a window.

The following are the 7 effects implemented:

Colour Tint - vertical colour gradient

Standard Blur - a basic blur effect

Underwater - colour tint paired with distortion to create an effect similar to being underwater

Gaussian blur - uses an algorithm and uses 2 pixel shaders one blurring vertically and the other horizontally to create a nicer natural blur

Bloom - Increases the lighting from the light source to generate a brighter scene

Hue gradient - a colour gradient that changes colour over time using the HSL colour space (Hue, Saturation, Luminance)

Retro - My personal favourite, Limits the colour palette and increases the size of the pixels to create a Retro game style effect

Melanoma Detection

Here is a piece of work that I was quite proud of, an AI system that was able to predict whether or not a picture of potential Melanoma was cancerous or not. The model,developed in Python was trained with the largest public database of melanoma images, The ISIC Database, this was a big challenge having little to no knowledge in actual AI, so alot of research was done to find an appropriate AI model that was lightweight, efficent and accurate. I opted to use the MobileNetV2 model, which was a model designed to perform well on mobile devices. The model was a pretrained model that was made for images making it perfect for this project. The model was trained using the ISIC training data and was tested using the ISIC testing data. The model was able to achieve an accuracy of 93.7% which was a great achievement. The model was then shown images that I had found that were not included in the ISIC database, and the model was able to predict whether or not the image was cancerous or not. I also included a probability value to show confident the model was with its prediction, a value close to 0.01 meant the image was most likely not cancerous, a value close to 0.99 meant the image was most likely cancerous and a value close to 0.5 meant the model was unsure.
